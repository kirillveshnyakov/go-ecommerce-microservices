# Frontend E-Commerce

Фронтенд для задания: каталог, корзина, чекаут, оплата/отмена заказа.

## Брендбук

- **#01A5E4** — основной синий (кнопки, ссылки)
- **#F4721E** — акцент оранжевый (CTA, цены)
- **#09111B** — фон
- **#151A1F** — карточки, поверхности

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173. В dev прокси в `vite.config.ts` разводит запросы: cart → 8080, product/stock/order (LOMS) → 8081.

Если запускаешь **preview** (`npm run build && npm run preview`), прокси не работает. Создай `.env` из `.env.example` и задай:
```bash
VITE_CART_URL=http://localhost:8080
VITE_LOMS_URL=http://localhost:8081
```
Затем снова `npm run build`.

## Функционал

- **User ID** хранится в cookie `user_id`, при первом заходе генерируется случайный. Кнопка **Сбросить** задаёт новый ID (для проверки «другого пользователя»).
- **Каталог** — товары по SKU 1–8 (и добавление по произвольному SKU). Кнопки +1, +2, +5 в корзину.
- **Корзина** — список позиций, удаление, кнопка «Перейти к оплате» (checkout).
- **Заказ** — после checkout переход на страницу заказа: статус, оплата, отмена.
